# M5-ThirdPartyAPIs-Scheduler


## Description

For this challenge we were tasked to add code with jQuery to an existing application, so that the HTML and CSS is dynamically updated by the Third-Party API. The application is a simple calendar application that allows a user to save events for each hour of the day and meets the following acceptance criteria: 
<ul>
<li>When the planner is opened the current day is displayed at the top.
<li>The user is presented with time blocks for standard business hours when scrolling down.
<li>When the user views the time blocks for the day, they will see that each time block is color coded to indicate whether it is in the past, present, or future.
<li>The user is able to enter an event when they click into a time block.
<li>When the save button for that time block is clicked then the text for that event will be saved in local storage.
<li>The saved events are displayed when the page is refreshed.
